Everyone pls go by the link - http://libgdx.badlogicgames.com/download.html and download the project Setup App
Use the app to create a new project with the name "my-gdx-game",

Also uncheck Android, iOS and HTML5 boxes, since we won't work with them.

Check Box2Dlights, Tools, AI and Box2D extensions (we might probably use them... or not... who knows?)

Then you can add all the files from this branch to existing folders.
